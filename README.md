# FAVO

FAVO (ou FinAgro Vale do Orvalho) é um MVP de um sistema FinTech e AgroTech aplicado o mundo de Stardew Valley. O objetivo do projeto é demonstrar a aplicação prática de TypeScript estrito no cálculo de algoritmos de Retorno sobre Investimento (ROI), projeção de fluxo de caixa sazonal e utilização de alguns conceitos essenciais para boas APIs RESTful, como contratos de dados rígidos entre AP e Client, princípios fundamentais do SOLID (como a Segregação de Responsabilidades e Inversão de Dependência), autenticação com API KEY e modelagem de dados.

## Tecnologias do Backend

- Linguagem: TypeScript
- Framework: Fastify
- Banco de Dados: PostgreSQL
- ORM: Prisma
- Validação: Zod

## Tecnologias do Frontend

- Linguagem: TypeScript
- Bundler: Vite
- Framework de Estilização: TailwindCSS
- Ícones: Lucide Icons (CDN)
