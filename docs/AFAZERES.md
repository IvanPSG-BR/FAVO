# Afazeres

- #F: Fácil, é uma implementação que pode ser considerada simples e rápida
- #M: Média, é uma implementação mais demorada, mas não necessariamente tão complexa
- #D: Difícil, é uma implementação que vai exigir um maior cuidado e atenção, consequentemente, tempo
- #?: Indefinido, é uma implementação que não fui capaz de categorizar
